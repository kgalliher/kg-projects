Edit Parcels

Instructions:

  • Create a new parcel record
  • Select one or more parcels
  • Option 1:
    • Copy the selected features to the parcel type (Copy Lines To)
    • Draw lines within the open boundary to form closed loops
    • Click Build Active Record
  • Option 2:
    • Digitize a closed loop of lines
    • Click Create Seeds to insert a seed
    • Click Build Active Record
  • Option 3:
    • Select 2 or more parcels
    • Click Merge Parcels
